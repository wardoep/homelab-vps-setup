Homelab VPS Setup

    Documentation for my personal homelab environment used to practice Linux administration, virtualization, remote access, and self-hosted tooling.

    Overview

    This repository documents the setup and ongoing use of my homelab environment, including Ubuntu Server, VirtualBox, Tailscale, and SSH-based workflows. I use this setup to build more hands-on experience with Linux systems, remote access, command-line tools, and the infrastructure that supports my personal technical projects.

    What I'm Using It For

    - Practicing Linux system administration
    - Working with Ubuntu Server in a virtualized environment
    - Using Tailscale and SSH for remote access and management
    - Learning more about CLI-based workflows
    - Supporting personal tools and assistant-related projects

    Skills Demonstrated

    - Linux administration
    - Virtual machine setup
    - SSH and remote access workflows
    - Tailscale configuration
    - CLI-based system management
    - Documentation and troubleshooting

    Why I Built It

    I wanted a hands-on environment where I could learn by doing instead of only reading about systems and infrastructure. This homelab gives me a place to practice technical skills that are relevant to IT support, system administration, and self-hosted tooling.

    Status

    Active project. I continue updating this repository as I improve the environment, document new workflows, and build more confidence with Linux and remote system management.
